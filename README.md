S14-TRABAJO PRÁCTICO EXPERIMENTAL_5

Este proyecto corresponde al Trabajo Práctico Experimental de la Semana 14.
Se desarrolló un backend utilizando Node.js, Express, MongoDB, JWT y OAuth con Google, cumpliendo con los requisitos establecidos por el docente:

CRUD completo para usuarios.

Conexión a base de datos MongoDB con Mongoose.

Autenticación mediante JWT (login tradicional).

Rutas protegidas con verificación de token.

Implementación de OAuth 2.0 (Google Login) usando Passport.

Pruebas realizadas en Thunder Client para validar el funcionamiento.

Este repositorio contiene la solución backend del trabajo práctico.
